# 🏢 ERP Automation Framework

## 🧪 Playwright + Python + BDD (pytest-bdd)

This repository contains a scalable **end-to-end test automation framework** built for an **ERP system**, focusing on the **Operations module and core business workflows**.

The framework is designed using **SDET best practices**, combining **Playwright for UI automation**, **pytest-bdd for Behavior Driven Development (BDD)**, and a **Page Object Model (POM)** structure for maintainability and reusability.

---

## 🚀 Key Features

- 🧩 BDD-based test design using Gherkin syntax  
- ⚙️ UI automation with **Playwright (Python)**  
- 🏗️ Scalable **Page Object Model (POM)** architecture  
- 🧪 Modular test structure for ERP workflows  
- 📊 Reusable test components and utilities  
- 🔁 CI/CD ready structure for pipeline integration  
- 📁 Organized test data management  
- 🐞 Clean reporting and debugging support

## 📌 Architecture

- `features/` → BDD feature files (Gherkin scenarios)  
- `steps/` → Step definitions (pytest-bdd bindings)  
- `pages/` → Page Object Models  
- `utils/` → Helpers & utilities  
- `conftest.py` → Playwright fixtures  
- `test_data/` → Test datasets
